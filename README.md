# 🛒 ShopSmart – Full Stack Grocery Platform

## 🚀 Overview
ShopSmart is a complete e-commerce-style grocery platform designed to handle real-world shopping workflows including authentication, product management, and admin control.

---

## 🧩 Core Modules
- 🔐 User Authentication (Login / Signup)
- 🛍️ Product Catalog
- ⚙️ Admin Dashboard
- 📦 Order Management

---

## 🛠️ Tech Architecture
Frontend → React  
Backend → Node.js + Express  
Database → MongoDB  

---

## 💡 Key Highlights
- Role-based access (Admin/User)
- REST API integration
- Scalable structure
